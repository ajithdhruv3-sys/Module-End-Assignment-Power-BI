# Module-End-Assignment-Power-BI
Bike Sharing System – Power BI Dashboard

This project analyzes bike availability, occupancy, capacity, and distribution across multiple cities and stations. The dashboard provides an overview of operational performance and helps understand how effectively the bike-sharing system is being utilized.

Project Overview
The dataset contains information about bike stations, available bikes, empty stands, total capacity, occupancy percentage, availability percentage, and contract locations. The Power BI report summarizes these metrics and visualizes trends across years, locations, and station names.

**Key Insights**
Overall System Metrics
Total Available Bikes: 20K

Total Empty Stands: 32K

Total Capacity: 57K

Occupancy Percentage: 34.34

Availability Percentage: 56.15

**Station-Level Insights**
Stations such as 10002, 10003, 112_AVDA, and 118_FCO show the highest number of available bikes.

Several stations consistently maintain moderate availability, indicating balanced usage.

Some stations show lower availability, suggesting higher demand or insufficient capacity.

**Yearly Trends**
Available bikes show a clear upward trend from 2022 to 2025.

2025 records the highest availability, indicating system expansion or improved operations.

**Occupancy vs Target**
Current occupancy is 34.34%, below the target of 50%.

The gauge visualization highlights a gap between actual and expected utilization.


**Capacity Utilization**
Total capacity is 57K, but only 20K bikes are currently available.

Empty stands (32K) indicate potential underutilization or imbalance in bike distribution.

The comparison chart highlights the gap between capacity and actual usage.

**Dashboard Features**
KPI cards for quick system overview

Bar charts showing station-level availability

Line chart showing yearly trends

Gauge chart comparing occupancy to target

Treemap showing bike distribution by contract location

Capacity vs availability comparison chart
